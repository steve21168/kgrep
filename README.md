# kgrep
```
NAME:
   kgrep - Used to grep keys in JSON or YAML

USAGE:
   output | kgrep [options] regexp

COMMANDS:
   help, h  Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --match_count value, -m value  The first n number (above 0) of matches to return (default: 0
   --help, -h                     show help (default: false)
```
